# Simple Interest Calculator

A Bash script that calculates simple interest based on user input for principal amount, rate of interest, and time period.

## Project Details
- Project Name: Simple Interest Calculator
- Author: Joseph Ritchwood
- Language: Bash
- Purpose: Calculate simple interest using the formula SI = (P * R * T) / 100

## Features
- Accepts user input for principal, rate, and time
- Validates numeric input
- Displays calculated simple interest
- Easy to run on any Unix/Linux system

## How to Use
1. Clone the repository
2. Make the script executable: chmod +x simple-interest.sh
3. Run the script: ./simple-interest.sh
4. Enter principal, rate, and time when prompted

## Example
Principal: 1000
Rate: 5
Time: 3
Simple Interest: 150
